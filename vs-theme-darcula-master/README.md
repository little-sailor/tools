# vs-theme-darcula
VisualStudio + R# (only VS2015) full color theme based on IntelliJ IDEA Darcula.
## Install
Through the VS plugin [plugin](https://marketplace.visualstudio.com/items?itemName=ms-madsk.ColorThemeDesigner)
